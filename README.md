# Implementations for CLRS

Implementations for some of the algorithms and data structures from
"Introduction to Algorithms" (abbreviated as CLRS) in **Python**.

## Table of Contents

- Algorithms:
  - Sorting:
    - Insertion sort
    - Selection sort
    - Bubble sort
    - Merge sort
    - Heapsort
    - Quicksort
    - Counting sort
  - Search:
    - Linear search
    - Binary search
- Data structures:
  - Binary heap
  - Queue
