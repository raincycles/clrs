# Implementations for CLRS

Implementations for some of the algorithms and data structures from
"Introduction to Algorithms" in **Python**.

## Table of Contents

- Sorting:
  - Insertion sort
  - Selection sort
  - Bubble sort
  - Merge sort
  - Heapsort
  - Quicksort
  - Counting sort
- Searching:
  - Linear search
  - Binary search
- Data structures:
  - Binary heap
  - Double-ended queue (Deque)
  - Linked list
  - Binary search tree
  - Hash table
